# docker-git-cicd-learning
Practical exploration of Git, Docker, CI/CD, DevOps and modern software delivery practices from a Technical Delivery and Agile perspective.
## Objectives
- Learn Git and GitHub workflows
- Learn Docker fundamentals
- Understand CI/CD concepts
- Build hands-on deployment experience
- Create a professional technical portfolio
## Learning Environment
- Rocky Linux 9
- Docker CE
- Git & GitHub
## Roadmap
- Git Fundamentals
- Docker Basics
- Dockerfile Creation
- Docker Compose
- GitHub Actions
- CI/CD Pipeline
- Demo Application Deployment
